# git-go
Git-go will help show you the *very* basics of git.

# Prework
[Install git](https://git-scm.com/downloads)

[Configure identity](https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup#Your-Identity)

# Usage
## Mac & Linux
```
> bash git-go.sh
```

## Windows
```
> git-go.bat
```
